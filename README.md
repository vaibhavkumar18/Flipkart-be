# 🧠 E-Commerce Backend API

Backend service for the E-Commerce application built using **Node.js, Express, and MongoDB**.

This API handles authentication, product management, cart operations, address management, and order processing.

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (Authentication)
- Nodemon (Development)

---

## 🚀 Getting Started


### 1. Clone Repository
```bash
git clone https://github.com/vaibhavkumar18/Ecom-backend.git
cd Backend
```

### 1.Create a Folder Named "Backend"
```bash
cd Backend
git clone https://github.com/vaibhavkumar18/Ecom-backend.git
```
---

### 2. Install Dependencies
```bash
npm install
```
---

### 3. Environment Variables (Backend)
#### Create .env file:
```bash
PORT=3000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
FRONTEND_URL = you_frontend_url
NODE_ENV="development"
```
---

### 4. Run Application
```bash
npm run dev
```
